## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/header-animated-react-native.git
```

Go to the project directory

```bash
  cd header-animated-react-native
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
